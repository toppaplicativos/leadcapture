# LeadCapture Connect (Android)

App **nativo** multi-slot: HOST / SIDECAR / WORK_PROFILE / VIRTUAL + disparo de conexões Baileys.

## Abrir

Android Studio → `apps/android-connect`  
`.env` → `API_BASE_URL=https://app.leadcapture.online/`

## Uso rápido

1. Login com conta do painel  
2. **Sync** → scan packages / +2 slots virtual  
3. Criar slot (escolha isolamento)  
4. Abrir slot → celular → **Criar instância + disparar pairing**  
5. Cole o código no WhatsApp nativo (Aparelhos conectados)

## Isolamento

| Mode | Uso |
|------|-----|
| HOST | WhatsApp / Business oficial |
| SIDECAR | Package de clone sideload |
| WORK_PROFILE | 2º perfil se existir |
| VIRTUAL | Pool de packages instalados |
